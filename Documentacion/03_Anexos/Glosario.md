# Anexo D — Glosario de Términos
## Proyecto: Sistema de Gestión de Cotizaciones — "InfleSusVentas"

> Este glosario se consolida **al final del proyecto**. Se irá alimentando desde los requisitos de
> dominio (Semana 9). Orden alfabético.

| Término | Definición |
|---|---|
| **Arco** | Inflable publicitario con forma de arco; se cotiza por largo y alto. |
| **Carpa** | Inflable tipo carpa; cuadrangular (alto, largo, ancho) o circular (diámetro, altura). |
| **Cotización estándar** | Cotización completa con validación de RUC y trazabilidad total. |
| **Cotización rápida** | Flujo simplificado, sin validación de RUC y con fecha a nivel de mes. |
| **Globo** | Inflable esférico; se cotiza por altura (el diámetro se deriva por proporción). |
| **IGV** | Impuesto General a las Ventas (18 % en Perú). |
| **Ítem** | Cada producto inflable incluido en una cotización. |
| **MoSCoW** | Método de priorización: Must / Should / Could / Won't have. |
| **Razón Social** | Nombre legal asociado a un RUC. |
| **RUC** | Registro Único de Contribuyentes. |
| **Skydancer** | Inflable danzante (bailarín de aire); se cotiza por altura. |
| **SRS / ERS** | Especificación de Requisitos de Software. |
| **Subtotal** | Suma de precios base (sin IGV). |
| **Tótem** | Inflable vertical publicitario; se cotiza por altura. |

> Añadir términos nuevos a medida que se identifiquen en el dominio.
